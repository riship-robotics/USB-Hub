## How to Open This Project
This project was built using EasyEDA Pro. To open and edit the PCB follow these steps.
1. Download the .epro file from this repository.
2. Open the EasyEDA Pro Editor: https://pro.easyeda.com/editor
3. In the top menu, go to File > Import >*EasyEDA Professional.
4. Select the .epro file to load the complete schematic and PCB layout.
